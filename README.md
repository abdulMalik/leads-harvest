# Leads Harvest

A Chrome extension that extracts local business contact data from Google Maps for sales prospecting and outreach.

> **Not on the Chrome Web Store.** Install it manually from the latest release below.

## Install (Chrome / Edge / Brave)

1. Download **`leads-harvest-v0.3.0.zip`** from the [latest release](../../releases/latest).
2. **Unzip** it to a folder you'll keep (don't delete it after — Chrome loads the extension from this folder).
3. Open `chrome://extensions` (or `edge://extensions`).
4. Turn on **Developer mode** (top-right toggle).
5. Click **Load unpacked** and select the unzipped folder.
6. Pin the Leads Harvest icon and open it on a `google.com/maps` search.

### Updating to a new version
Download the newer zip, unzip it (overwriting the old folder), then click the **reload** icon on the extension card at `chrome://extensions`.

## Usage

1. Go to [google.com/maps](https://www.google.com/maps) and search for a business type + location.
2. Open the Leads Harvest popup and click **Start Harvesting**.
3. Keep the Maps tab in front while it runs.
4. Export your leads to **CSV** or **JSON**.
