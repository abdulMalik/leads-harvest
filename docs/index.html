Leads Harvest

Turn Google Maps into a lead list

Extract local-business contacts — name, phone, website, email, address, hours, ratings, and social profiles — and export to CSV or JSON in a couple of clicks.

⬇ Download the latest release Free Chrome / Edge / Brave extension · manual install (steps below)

Install in 6 steps

  1. On the latest release page, download leads-harvest-*.zip.
  2. Unzip it to a folder you'll keep — don't delete it afterward; Chrome loads the extension from this folder.
  3. Open chrome://extensions (or edge://extensions).
  4. Turn on Developer mode (top-right toggle).
  5. Click Load unpacked and select the unzipped folder.
  6. Pin the Leads Harvest icon, open Google Maps, search, and click Start Harvesting.
Why a manual install? Leads Harvest is distributed off the Chrome Web Store, so you load it yourself via Developer mode. To update later, download the newer zip, unzip over the old folder, and click the reload icon on the extension card.

What it captures

Contact details

Phone, website, and email (discovered from each business's own site).

Social profiles

Facebook, Instagram, X, LinkedIn, YouTube, and TikTok.

Business context

Category, rating, review count, address, and full per-day hours.

Clean export

CSV, Excel, or JSON in a sales-priority order, with selectable fields and de-duplicated rows.

Fast mode

Skip email/social lookup for raw speed, or keep it on for full enrichment — your choice.

Live preview

See harvested leads and counts in the popup before you export.
